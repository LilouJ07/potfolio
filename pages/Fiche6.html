Encodage d'un texte:

Code ASCII:

ASCII=American Standart Code for Infrrmation Interchange

ASCII utilise 7 bits (128 carcs total (0 à 127))

    0 à 31 carctères de contrôlie (faire actions)
    48 à 57 sont les chiffres
    65 à 90 sont les majuscules
    97 à 122 sont es minuscules

Les derniers sont les accolades, tildes,...

ord('k')=donne le code ASCII de k, ch(84)=donne la lettre correspondant a 84 soit T

ISO-8859:

8 bits donc trop petit (peut pas écrire arabe, chinois,...)

Unicode:

0xxxxxxx = 1 octet codant 1 à 7 bits
110xxxxx 10xxxxxx = 2 octets codant 11 bits
...

Chaines de caracteres:

Elles sont entre " ", 1ere lettre=indice 0

    mot.upper() renvoie chaine mot en Majuscule
    mot.lower() renvoie chaine mot en minuscule
    str(nb) renvoie chaine de carac contenant écriture décimale du nb
    int(ch) transforme chaine en valeur numérique entière
    float(ch) transforme chaine en valeur numérique flottante
    ch.count(texte) compte le nombre de fois où texte est présent dans ch
    ch.replace(txt1,txt2) remplace txt1 mar txt2 chaque fois qu'il apparait dans ch
    ch.find(texte) indique la position dans la chaine ch où se trouve texte, renvoie -1 si texte nn'y est pas
