Les systèmes d'exploitation:

Le système dd'exploitation traduit le langage humain en langage machine pour que l'ordinateur comprenne et éxécute l'action.

Il peut:

    - fournir un interface entre l'homme et la machine
    - gérer les resources cce l'ordinateur (mémoire, processeur, périphériques, gestion d'énergie)
    - gérer les utilisateur + leur fichier + leurs droits d'accès
    - être indépendant du matériel (un même OS peut être exécuté sur des ordinateurs n'ayant pas les mêmes composants)

Il existe 2 grandes familles de systèmes d'exploitation : Windows / Unix

Principaux systèmes d'exploitation rencontrés: Windows, MacOS, Linux, Android, iOS.

Windows:

- Système d'exploitation propriétaire -> code source appartient uniquement à la société (Microsoft)

De plus il est strictement encadrée par sa license + son code source = pas accessible + pas dristribuable + pas modifiable

GNU/Linux, macOS,iOS, Android:

On peut y retrouver des lignes de commande

Unix/Linux =systèmes d'exploitation libres -> code source accessible publiquement ( utilisable + modifiable selon les termes d'une licence de logiciel libre)

Les fichiers:

Afin de stocker les données, pour Linux et Unix le système de fichiers est composé de répertoires/dossiers + fichiers.

-> Former sous formes d'arborescence
-> Bien nommer les fichier
-> Possible de les gérer avec lignes de commandes
-> Répertoire Racine ( / ): contient tout répertoires (usr, ect, dev (obligatoirs + contiennent une partie des données nécessaires au fonctionnement du système d'exploitation)
-> Répertoire home = répertoires personnels des utilisateurs (ou aussi sous le nom user)
-> Pour accéder à un fichier ou un répertoire: chemin d'accès absolu / chemin d'accès relatif

Le chemin d'accès absolu:

Chemin qui part de la racine / suivi de la suite des noms de répertoire jusqu'a la destination.
Il est indépendant du répertoire où on est.
ex: /home/Angie

Le chemin d'accès relatif:

Chemin qui tient compte du répertoire courant (là où on se situe) + donne la liste des noms du ou des répertoires à partir du répertoires ( du répertoire courant à la destination)
ex: Angie si le répertoire est dans le répertoire home et qu'on se situe dans home

. = répertoire courant, ..= répertoire père du répertoire courant (permet de remonter d'un niveau dans l'arborescence des fichier)

Lignes de commande:

Chaque système d'exploitation possède un terminal (sous programme permettant l'exécution des taches en écrivant les commandes)

Pour lancer le terminal:
- Windows = touche windows + R et en rentrant cmd
- Ubuntu(Linux) = icone sous forme d'écran noir - macOS: dans le Finder, Applications, Utilitaires et Terminal (écran noir)

Quelques commandes communes:

- tree = visualiser une arborescence de fichiers
- pwd = connaitre le répertoire courant
- ls = liste les fichiers + répertoire du répertoire courant
- cd = Change de répertoire (ex: cd rep1\rep2)
- rm = Supprimme un fichier ou un répertoire (ex: rm fichier.txt)
- man "commande" = aide pour la commande (ex: man ls)
- mkdir = crée un répertoire (ex: mkdir essai.txt)
- cat = affiche un fichier dans la console (ex: cat fichier.txt)

Quelques commandes différentes:

Windows / Linux et macOS
- copy / cp = copie fichier dans un autre endroit ou sous un autre nom)
- move / mv = déplace un fichier
- ren / mv = renomme un fichier
- / rm = supprime un dossier/fichier

Droits et permissions:

Chaque fichier ou répertoire a des proprités (droits d'accès et de permission)

Les différents droits: droits d'écriture, de lecture, d'exécution

Pour les modifier on peut utiliser l'interface graphique ou la commande en ligne chmod.

Droits de lecture (r - read), d'écriture (w - write), d'exécution (x - execute)

Ces différents droits peuvent être associés à 3 classes d'utilisateur:
- propriétaire du fichier (user (u))
- groupe auquel appartient le fichier (group (g))
- tous les autres (other (o))

La commande chmod admet 4 paramètre:
- 1er précise la cible: u,g,o, a pour tous
- 2eme + ou - suivant si on donne ou enlève les droits
- 3eme r,w,x les permissions concernées
- 4eme le fichier ou répertoire concerné

-R après chmod = que les autorisations sont modifiées pour tout le contenu d'un dossier.

On peut modifier les droits valeur octale (base 8) de 3 chiffres.

rwx = 111 = 7
rw- = 110 = 6
r-x = 101 = 5
r- = 100 = 4
-wx = 011 = 3
-w- = 010 = 2
-x = 001 = 1

Ex: chmod 644 Saison = chmod u=rw, go=r Saison
