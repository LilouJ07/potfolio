Les Tableaux:

Tableaux = listes (python) -> stock valeurs ou données dans une variable
Valeurs/données = éléments(elem) ou items de la liste.
Les tableaux sont nutables/muables = modifiable après création.
Le premier élément de la liste = indice 0.

Importantes Instructions:

    L=[e1,e2,...,en] = création liste L contenant éléments e1,...
    L=[] = liste vide
    L[i] = renvoie elem d'indice i
    L[-1] = renvoie dernier elem de L
    L[i]=elem = affecte à l'élément d'indice i l'élément elem
    len(L) = renvoie la longueur de L
    e in L = teste si e est dans L

    L.append(elem) = rajoute elem (à la fin de L)
    L.insert(j,elem) = insère elem au rang j
    L.remove(elem) = surprime le 1er elem dans L
    L.pop(i) = supprime elem d'indice i dans L
    [1,2]+[3+4] = concatène
    L.reverse = inverse
    L.count(elem) = compte les occurences de elem
    L.index(elem) = trouve indice de elem
    min(L), max(L) = trouve le plus petit, "" le plus grand
    choice(L) = Choisit au hasard elem dans L (utilisation module Random)
    L[ :m] = affiche les m premiers éléments de L
    L.split(sep) = prend chaque mots et les sépare pour en faire une liste avec comme éléments les mots list(ch) = prend chaque lettre et les sépare pourr en faire une liste comme éléments les lettres

    L[0],L[5]=L[5],L[0] cela inverse les valeurs 0 avec la 5 et la 5 avec 0

Les Tuples:

    *Collection ordonnée d'éléments
    *Note avec des () séparés par ,
    *Peut être affecté à une variable et une fonction peut retourner un tuple

1er élément=indice 0, un tuple à 1 seul élement doit avoir une , après

Extraire éléments: Utiliser affectation multiple ou indexation
Affectation multiple permet d'échanger les valeurs de 2 variables sans passer par une variable

ex:coordonnees=(15,56) , coordonnees[1]=56
a=3 , b=5, a,b=b,a=(5,3)

Il peut y avoir des Tuples dans d'autres Tuples.

Propriétés:

    *Un Tuple est immuable (1 fois construit on peut pas modifier contenu)
    *t=() tuple vide
    *Fonction utilisables: len,+,*,in
    *On ne peut pas ajouter un élément entre le 1er et le dernier (seulement au début et à la fin)

Les Dictionnaires:

Un dictionnaire est une sorte de liste où on utilise des clés (valeurs), on remplace les [ ] par des { }.

Les dicos sont muables ou mutables.
dico={"Nom":Jouanolou} Nom-> clé, Jouanolou->valeur associée à clé

Opération:

    - dico.keys() renvoie toutes les clés du dico.
    - dico.values renvoie toutes les valeurs du dico
    - dico.items renvoie tous les éléments de dico
    - dico.get(...) renvoie la valeur associée à la clé rentrée
    - if "..." in dico: dire si il y est
    - Parcourire le dico: for cle in dico:
    - dico.copy() copie le dico
    - del dico["computer"] supprime la clé et sa valeur
